== Frequently Asked Questions ==

= Is this calculator medically accurate? =

The calculator provides estimates based on standard pregnancy calculations (280 days from the first day of the last menstrual period). However, these are estimates only. Every pregnancy is unique, and only a healthcare provider can provide an accurate due date through ultrasound and examination.

= Can I translate the calculator to my language? =

Yes! The plugin comes with 17 pre-translated languages, and you can translate it to any language using:
* WordPress language settings (Settings â†’ General â†’ Site Language)
* Translation plugins like TranslatePress, WPML, or Polylang
* Manual .po/.mo file translation using Poedit

= Does it work with page builders? =

Yes! Use the shortcode `[pregnancy_due_date_calculator]` with:
* Elementor
* Divi Builder
* WPBakery
* Beaver Builder
* Any other page builder that supports shortcodes

= Can I customize the colors and styling? =

Absolutely! The Gutenberg block provides 40+ customization options including:
* Colors (background, text, buttons, progress bar)
* Typography (font families, sizes, colors)
* Spacing (padding, margins, border radius)
* Layout (alignment, max width)

= Is it mobile responsive? =

Yes! The calculator is fully responsive and works perfectly on all devices including phones, tablets, and desktops.

= Does it store any user data? =

No. All calculations happen in the user's browser. No data is sent to external servers or stored in your database. The plugin is GDPR compliant.

= Can I add it to a widget area? =

Yes! Use the shortcode `[pregnancy_due_date_calculator]` in any widget area that supports shortcodes.

= Does it work with multisite? =

Yes! The plugin is multisite compatible and can be activated network-wide or on individual sites.

= How do I change the language? =

Go to Settings â†’ General â†’ Site Language and select your preferred language. The calculator will automatically display in that language if it's one of the 17 supported languages.

= Can I customize the disclaimer text? =

Yes! In the block settings sidebar, you'll find a "Note Section" where you can customize the disclaimer text or hide it completely.

== Medical Disclaimer ==

**Important:** This calculator provides estimated dates only based on standard pregnancy calculations. Pregnancy dating should always be confirmed by a healthcare professional through ultrasound examination and clinical assessment.

This tool is for informational purposes only and does not constitute medical advice, diagnosis, or treatment. Always consult with your doctor, midwife, or healthcare provider for:

* Confirmation of pregnancy
* Accurate due date determination
* Prenatal care guidance
* Any medical concerns or questions

Every pregnancy is unique, and actual delivery dates can vary significantly from calculated estimates.
